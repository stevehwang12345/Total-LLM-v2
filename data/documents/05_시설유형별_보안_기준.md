# 시설 유형별 물리보안 적용 기준

## 1. 목적
본 문서는 시설 유형에 따른 물리보안 적용 기준을 정의한다. 통합방위법, 범죄예방 건축기준 고시, ISMS-P 인증기준에 근거한다.

## 2. 대규모 시설 (공항, 항만, 발전소, 데이터센터)

### 2.1 법적 근거
- **통합방위법 시행령 제32조**: 국가중요시설 경비·보안·방호 규정
- **경비업법 제14조**: 특수경비원 배치 의무 (국가중요시설)
- **주요정보통신기반시설 관리 안내서 2026** (KISA/과기부)

### 2.2 심층방어 전략 (Defense-in-Depth, 4D)
| 단계 | 목적 | 수단 |
|------|------|------|
| **Deter (억제)** | 침입 시도 자체를 억제 | 경고 표지판, 조명, 가시적 CCTV, 울타리 |
| **Detect (탐지)** | 침입 시도를 조기 탐지 | 센서, CCTV AI 분석, 출입통제 모니터링 |
| **Delay (지연)** | 침입자 진행을 지연 | 강화 울타리, 이중문(Man-trap), 잠금장치 |
| **Deny (거부)** | 최종 접근 거부 | 금고, 보안 케이지, 경비 인력 대응 |

### 2.3 필수 보안 시스템
- 외곽 경계: 울타리 + 광섬유/진동 감지 + CCTV (열화상 포함)
- 출입구: 차량 차단기 + 번호판 인식 + 보안 검색대
- 건물 내부: 카드+생체인식 이중 인증 + CCTV + 침입감지
- 통제실: 24/7 상시 운영, 이중화 시스템, 비상 발전기
- 데이터센터 추가: 소방청 SOP 239 적용, 수냉식 소화설비, 케이블 보호

## 3. 일반 상업건물 (오피스, 쇼핑몰, 병원)

### 3.1 법적 근거
- **범죄예방 건축기준 고시** (국토교통부고시 제2021-930호)
- **ISMS-P 인증기준 2.4** (물리보안) — 정보통신서비스 제공자 의무
- **소방청 SOP 212** (일반건축물화재 대응절차)

### 3.2 보안 적용 기준
| 구역 | CCTV | 출입통제 | 침입감지 | 비고 |
|------|------|---------|---------|------|
| 로비/1층 | 4K, Validate급 | 카드 인증 | 영업시간 외 PIR | 방문자 관리 |
| 사무 층 | 2MP, Discern급 | 카드 인증 | 야간 PIR | 층별 접근 제한 |
| 서버실 | 4K, Scrutinize급 | 카드+생체 | 24시간 듀얼테크 | 투퍼슨룰 |
| 주차장 | 4MP, Characterize급 | 차량번호인식 | 울타리 센서 | 조명 확보 |
| 비상구 | 2MP | 비상시 자동 개방 | 개방 감지 | 페일세이프 |

### 3.3 병원 특수 요구사항
- 응급실: 24시간 CCTV + 비상호출 버튼
- 약품 보관실: 카드+생체 이중 인증, 출입 기록 5년 보존
- 신생아실: 전자 태그 기반 유아 위치 추적, 무단 반출 경보

## 4. 주거단지 (아파트, 공동주택)

### 4.1 법적 근거
- **범죄예방 건축기준**: 500세대 이상 공동주택 의무 적용
- **주택건설기준 등에 관한 규정**: CCTV 설치 의무 (승강기, 주차장 등)
- **개인정보보호법 제25조**: 영상정보처리기기 설치·운영 기준

### 4.2 CPTED (환경설계를 통한 범죄예방)
- **자연적 감시**: 투시형 담장, 1층 필로티 시야 확보, 조경 높이 제한
- **자연적 접근통제**: 출입구 최소화, 보행 동선과 차량 동선 분리
- **영역성 강화**: 바닥 패턴·색상으로 공적/사적 공간 구분
- **활동 지원**: 어린이놀이터, 운동시설을 단지 중앙에 배치 (자연 감시)

### 4.3 필수 보안 설비
- 단지 출입구: 차량 차단기 + 비디오 인터콤
- 동 출입구: 카드 또는 비밀번호 인증 + CCTV
- 승강기 내부: CCTV (사각지대 없는 각도)
- 지하주차장: CCTV + 비상벨 (50m 간격) + 조명 (50 lux 이상)
- 경비실: 통합 모니터링 시스템

## 5. 핵심 인프라 (전력, 수도, 통신)

### 5.1 법적 근거
- **국정원 지능형 전력망 사이버보안 가이드라인** (2026.03.11)
- **주요정보통신기반시설 관리·물리적 취약점 분석 안내서 2026** (KISA)
- **통합방위법 시행령 제32조**

### 5.2 보안 요구사항
- 제어 시스템과 업무 시스템 간 물리적/논리적 망 분리
- 외곽 다중 방어선 (울타리 → 감지 → 경비 → 내부 통제)
- 무인 시설: 원격 모니터링 + 이상 감지 시 자동 알림
- 자연재해 대비: 내진, 방수, 내풍 설계
- 전원 이중화: 상용 전원 + 비상 발전기 + UPS
